# GoalDoc

## Project Goal

Build a simple Pomodoro/routine system that helps track focused work sessions, recurring tasks, reminders, and progress.

## Current Scope

- Store Pomodoro sessions.
- Support simple daily planning.
- Track recurring weekly, monthly, and half-yearly tasks.
- Keep project decisions and progress visible.

## Success Criteria

- The project can be run and tested locally.
- Project documents are kept in GitHub.
- ChatGPT can read the documents when asked.
- Progress is tracked clearly enough to resume after a break.
